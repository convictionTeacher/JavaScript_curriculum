returnを使って関数の処理結果を呼び出し元に返し、その値をさらに活用できます。

正解は「完了しました」です。

Step7: アロー関数

関数式をより短く書けるアロー関数の書き方に慣れ、モダンなコードを書けるようになります。
menu_book

提出は必要ありません。コードに触れて慣れてみましょう。

Q1: コンソールに表示される内容として正しいものを選んでください。

const greet = () => {
  console.log("こんにちは！");
};
greet();
選択肢を選んでください。 〇 正解！ ✕ 不正解。
正解は「こんにちは！」です。
() => { }はアロー関数の書き方で、greet()で呼び出せます。

Q2: コンソールに表示される内容として正しいものを選んでください。

console.log(double(6));
選択肢を選んでください。 〇 正解！ ✕ 不正解。
正解は「12」です。
(n) => n * 2 は省略形で、n * 2の計算結果を自動的に返します。6×2=12が表示されます。

提出は必要ありません。コードに触れて慣れてみましょう。

};

〇 正解！ ✕ 不正解。
正解は ①=> です。
() => { }の形がアロー関数です。=>（イコールと大なり記号）を使います。

Q2: アロー関数の引数名の空欄を埋めてください。

const square = () => n * n;
console.log(square(4));
〇 正解！ ✕ 不正解。
正解は ①n です。
引数nに4が渡され、n*n=16が返されます。
