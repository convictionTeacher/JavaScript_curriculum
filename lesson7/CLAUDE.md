step1 関数の定義
function キーワードで関数を書く
例：function greet() { … }

step2 関数の呼び出し
作った関数を呼ぶコードを書く
例：greet();

step3 引数あり関数
関数に値を渡して処理する
例：名前を渡して挨拶する関数

step4 戻り値（return）
計算結果や値を返す関数を書く
例：合計を返す関数

step5 複数引数
2つ以上の値を受け取る関数を書く
例：2つの数の計算

step6 関数式
const fn = function() { … } の形を書く
変数に関数を代入して使う練習

step7 アロー関数
const fn = () => { … } の形を書く
短く書く方法に慣れる

step8 コールバック関数
配列メソッドに渡す関数を書く
例：forEach / map に渡す簡単な処理
